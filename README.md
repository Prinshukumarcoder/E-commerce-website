### Run Locally

To run **Glowing** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithPrinshuCoder/glowing.git
```

Windows:

```bash
git clone https://github.com/codewithsPrinshuCoder/glowing.git
```
